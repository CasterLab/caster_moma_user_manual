# CasterMoma软件指南

本手册用于描述CasterMoma机器人的软件使用方法，请在阅读本文前，确保阅读过《CasterMoma用户手册》

## 注意事项

1. 机械臂活动范围大，请在使用前，清空CasterMoma机器人的操作空间，避免磕碰到其它设备
2. 机械臂在关机/异常断电后，会自由下落。所以在不使用机械臂时，尽可能将机械臂收起
